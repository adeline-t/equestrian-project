## ğŸ”„ Common Workflows

### Creating a New Rider-Horse Pairing

1. Navigate to a rider's profile
2. Click "Add Horse Pairing"
3. Select a horse from the dropdown
4. Set pairing start date (optional)
5. Save the pairing

### Viewing Pairing History

1. Open a rider or horse profile
2. View the "Pairings" section
3. See current and past pairings
4. Filter by date range if needed

### Ending an Pairing

1. Navigate to the pairing
2. Set an end date
3. Save changes
4. Pairing moves to "Past Pairings"

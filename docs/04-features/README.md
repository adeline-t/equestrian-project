## ğŸ”„ Common Workflows

### Creating a New Rider-Horse Association
1. Navigate to a rider's profile
2. Click "Add Horse Association"
3. Select a horse from the dropdown
4. Set association start date (optional)
5. Save the association

### Viewing Association History
1. Open a rider or horse profile
2. View the "Associations" section
3. See current and past associations
4. Filter by date range if needed

### Ending an Association
1. Navigate to the association
2. Set an end date
3. Save changes
4. Association moves to "Past Associations"
